# Home Lab Security Analysis

This repository documents a practical cybersecurity home lab project conducted using Kali Linux. The project focuses on network enumeration, packet analysis, threat intelligence investigation, and vulnerability assessment using industry-standard tools such as Nmap, Wireshark, VirusTotal, AbuseIPDB, and DVWA.

The objective of this project is to develop hands-on SOC Analyst and cybersecurity skills by identifying network assets, analyzing traffic patterns, investigating external IP addresses, and assessing common web application vulnerabilities in a controlled environment.
